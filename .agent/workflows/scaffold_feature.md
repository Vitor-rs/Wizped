# Scaffold Feature Workflow

**Referência**: Siga o "Feature Creation Workflow" definido em `.agent/rules/02-standards.md`.

**Passos**:
1. Leia o arquivo `.agent/rules/02-standards.md`.
2. Execute o comando `npm run generate feature` no terminal.
3. Siga rigorosamente os passos de registro de IPC e Segurança descritos na regra.
4. Execute `npm run lint` para validar.
