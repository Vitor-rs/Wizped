# General Principles

1. **Robustness Over Speed**: Do not implement "hacky" fixes ("gambiarras"). Solve the root cause.
2. **Strictness**: Types must be explicit. `any` is forbidden unless strictly necessary and documented.
3. **Agent-First**: Automated tools (Plop, Scripts) take precedence over manual creation.
4. **Documentation**: All architectural decisions must be reflected in these rules.
